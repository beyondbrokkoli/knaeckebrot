## Acknowledgments & Credits

This engine is powered by the following incredible open-source technologies:

* **[LÖVE (love2d)](https://love2d.org/):** An awesome framework for 2D games in Lua. LÖVE provides the windowing, input, and canvas API that makes drawing our software buffer possible. (Licensed under the zlib/libpng license).
* **[LuaJIT & FFI](https://luajit.org/):** Created by Mike Pall. The sheer speed of this engine is entirely thanks to LuaJIT's Just-In-Time compiler and the FFI (Foreign Function Interface) library, which allows us to write C-level memory structures directly in Lua. (Licensed under the MIT License).
